# ShadCN to Fluid Templates

> Extension key: `shadcn2fluid_templates`
>
> Version: `2.0.0`
>
> TYPO3 target: `^14.2@dev`
>
> Author: `webconsulting studio`

## Overview

This extension provides shadcn/ui-inspired content elements for TYPO3 14, implemented as Content Blocks and rendered with Fluid templates. It is v14-only and site-set-first.

## Highlights

- TYPO3 `^14.2@dev`
- Content Blocks `^2.1`
- Site set for CSS and TypoScript
- Workspace-aware content staging for `tt_content`
- External chart JavaScript asset instead of inline script blocks
- PHPStan level 9 and PHPUnit coverage for the shipped PHP/configuration layer

## Included content elements

- Hero
- Accordion / FAQ
- Alert / Callout
- Card
- Chart
- Compare
- Contact
- Call to Action
- Feature Grid
- Footer
- Gallery / Portfolio
- Logo Cloud
- Navbar
- Pricing
- Stats
- Team
- Testimonial
- Text with Media
- Timeline
- Blog Teasers
