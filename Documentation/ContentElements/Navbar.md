# Navbar

**Content Block Name:** `shadcn2fluid/navbar`

Navbar provides a marketing-style navigation bar with brand, navigation items, and CTA buttons.

## Templates

- `full-width`
- `two-columns`
- `stacked`

## Main fields

| Field | Type | Purpose |
|-------|------|---------|
| `layout` | Select | Navbar layout template |
| `brand` | Text | Brand text |
| `brand_link` | Link | Optional home link |
| `nav_items` | Collection | Label and link pairs |
| `secondary_button_*` | Mixed | Secondary action |
| `primary_button_*` | Mixed | Primary CTA |
