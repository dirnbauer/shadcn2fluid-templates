# Footer

**Content Block Name:** `shadcn2fluid/footer`

Footer provides brand text plus two configurable link groups for navigation and utility links.

## Templates

- `full-width`
- `two-columns`
- `stacked`

## Main fields

| Field | Type | Purpose |
|-------|------|---------|
| `layout` | Select | Footer layout template |
| `brand` | Text | Brand or company name |
| `description` | Textarea | Short brand description |
| `copyright` | Text | Copyright text |
| `footer_links` | Collection | Main footer navigation |
| `utility_links` | Collection | Secondary/legal links |
