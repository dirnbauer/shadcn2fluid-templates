# Archived

`webconsulting/shadcn2fluid-templates` has been merged into
[`webconsulting/desiderio`](https://github.com/dirnbauer/desiderio) starting
with Desiderio 2.0.0.

This extension is **no longer maintained**. New work, bug fixes, and new
content elements happen in the Desiderio repository. The `s2f:` Fluid
namespace has been retired; the same components are available under the
`d:` (Desiderio) namespace.

If you depend on `webconsulting/shadcn2fluid-templates`, migrate to Desiderio
2.0 — there is no backward-compatibility shim.

Last release of this extension: [`v3.0.0`](https://github.com/dirnbauer/shadcn2fluid-templates/releases/tag/v3.0.0-archived).
